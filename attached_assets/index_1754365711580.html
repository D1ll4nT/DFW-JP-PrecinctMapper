About

This tool uses official precinct boundary data to determine which Justice of the Peace precinct contains a given address. The data is sourced from the North Texas Evictions Project and represents the exact precinct boundaries as provided by each county.

Note: Geocoding is performed using OpenStreetMap's free Nominatim service. For best results, include the city and state in your address.
